# Nano Banana Studio - 设计头脑风暴

## 项目概述
一个 AI 绘图网站，用于调用 Nano Banana Pro 模型，兼容 OpenAI、Gemini、Claude 格式的 API。用户可以自定义 API 端口，丰富的自定义选项，可爱且现代化的界面，大量动效。

## 方案一：Kawaii Bubble Pop（卡哇伊气泡风格）

<text>
**Design Movement**: Neo-Kawaii + Glassmorphism 融合风格

**Core Principles**:
1. 圆润柔软的形态语言 — 所有元素都带有大圆角和柔软的边缘
2. 半透明层叠 — 毛玻璃效果创造深度和梦幻感
3. 活泼的微交互 — 每个操作都有弹性反馈
4. 糖果色调的温暖感 — 让技术工具感觉亲切可爱

**Color Philosophy**: 
- 主色：蜜桃粉 (#FFB5A7) — 温暖、亲切、有食欲感（呼应 Banana 主题）
- 辅色：香蕉黄 (#FFE66D) — 活力、创造力、品牌识别
- 强调色：薰衣草紫 (#C9B1FF) — 神秘、AI 的智慧感
- 背景：奶油白到淡粉渐变 — 柔和不刺眼
- 深色模式：深紫蓝 (#1A1333) 搭配霓虹糖果色

**Layout Paradigm**: 
- 左侧固定侧边栏（API 配置 + 历史记录）
- 中间主画布区域（生成结果展示）
- 右侧可折叠面板（绘图参数调节）
- 底部浮动工具栏（快捷操作）

**Signature Elements**:
1. 漂浮的气泡粒子背景动画
2. 卡通香蕉吉祥物在各处出现
3. 波浪形分割线和云朵形状的卡片

**Interaction Philosophy**: 弹性物理动画 — 按钮按下有弹跳感，面板展开像气泡膨胀，加载时有可爱的跳动动画

**Animation**: 
- Spring physics 弹性动画用于所有过渡
- 气泡上浮粒子效果
- 卡片 hover 时轻微浮起 + 光晕
- 加载时香蕉旋转动画
- 面板展开/收起的果冻效果

**Typography System**: 
- 标题：Fredoka One — 圆润可爱的显示字体
- 正文：Nunito — 圆润但可读性强的正文字体
- 代码/技术：JetBrains Mono — 清晰的等宽字体
</text>

## 方案二：Neon Synthwave Studio（霓虹合成波工作室）

<text>
**Design Movement**: Synthwave + Cyberpunk 赛博朋克风格

**Core Principles**:
1. 深色背景上的霓虹光效 — 高对比度的视觉冲击
2. 网格线和扫描线纹理 — 复古未来主义
3. 光芒四射的交互反馈 — 每次操作都像激活了什么
4. 渐变色的流动感 — 色彩永远在运动

**Color Philosophy**:
- 主色：电光蓝 (#00D4FF) — 科技感、未来感
- 辅色：霓虹粉 (#FF2D95) — 活力、创造力
- 强调色：电光紫 (#BD00FF) — 神秘、力量
- 背景：深空蓝黑 (#0A0A1A) — 深邃、沉浸
- 发光效果：所有主色都带 glow 效果

**Layout Paradigm**:
- 全屏沉浸式布局
- 顶部命令栏（类终端风格）
- 左侧垂直图标导航
- 中心区域分屏：左边参数，右边预览
- 浮动的状态指示器

**Signature Elements**:
1. 扫描线 + 网格背景动画
2. 霓虹边框发光效果
3. 像素化的香蕉 Logo

**Interaction Philosophy**: 光效反馈 — hover 产生光晕扩散，点击产生涟漪光效，拖拽留下光轨

**Animation**:
- CSS glow pulse 呼吸灯效果
- 背景网格缓慢移动
- 文字打字机效果
- 霓虹闪烁过渡
- 数据流粒子效果

**Typography System**:
- 标题：Orbitron — 几何化的未来感字体
- 正文：Space Grotesk — 现代几何无衬线
- 代码：Fira Code — 带连字的等宽字体
</text>

## 方案三：Banana Garden（香蕉花园 — 自然有机风格）

<text>
**Design Movement**: Organic Modernism + Soft UI (Neumorphism 2.0)

**Core Principles**:
1. 自然有机的形态 — 不规则的圆角、流动的曲线
2. 柔和的光影层次 — 内凹外凸的触感设计
3. 植物/水果主题的装饰元素 — 让 AI 工具充满生命力
4. 呼吸感的留白 — 宽松舒适的间距

**Color Philosophy**:
- 主色：成熟香蕉黄 (#FFDA6B) — 温暖、成熟、品牌核心
- 辅色：叶子绿 (#7EC8A0) — 自然、生长、创造
- 强调色：热带珊瑚 (#FF8A80) — 活力、热情
- 背景：暖米色 (#FFF8F0) — 自然纸张质感
- 深色模式：深森林绿 (#1B2B22) 搭配金色高光

**Layout Paradigm**:
- 顶部导航栏（带有机形状装饰）
- 主区域采用卡片式瀑布布局
- 侧边抽屉式参数面板（从右侧滑出）
- 底部状态栏（生成进度、API 状态）

**Signature Elements**:
1. 有机 blob 形状的背景装饰
2. 手绘风格的图标和分割线
3. 叶子/藤蔓装饰元素在角落

**Interaction Philosophy**: 自然流动 — 元素像植物生长一样展开，过渡像水流一样平滑，反馈像果实成熟一样渐变

**Animation**:
- Blob 形状的缓慢变形动画
- 卡片出现时的生长动画
- 加载时的种子发芽动画
- 面板滑动的弹性曲线
- 背景装饰的轻微浮动

**Typography System**:
- 标题：Quicksand — 圆润现代的几何字体
- 正文：DM Sans — 清晰友好的正文字体
- 代码：Source Code Pro — 经典等宽字体
</text>

## 最终选择：方案一 — Kawaii Bubble Pop

选择理由：
1. 最符合"可爱且现代化"的需求
2. Glassmorphism 效果天然适合大量动效
3. 糖果色调与 Banana 品牌完美契合
4. 弹性物理动画能创造丰富的交互体验
5. 三栏布局最适合 AI 绘图工具的工作流程
